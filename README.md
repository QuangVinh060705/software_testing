- 5/1 : ảnh điểm làm trên web https://cantunsee.space/, web chỉ ra và luyện xem giao diện hợp lý và đẹp mắt, giống nhất với thực tế
- 8/1 : Dự án nhằm thực hành viết Unit Test với JUnit 5 thông qua bài toán phân tích điểm số học sinh.
        Chương trình gồm lớp StudentAnalyzer dùng để đếm số học sinh giỏi (≥ 8.0) và tính điểm trung bình hợp lệ.
        Các điểm < 0 hoặc > 10 được xem là dữ liệu không hợp lệ và bị loại bỏ.
        JUnit được sử dụng để kiểm thử các trường hợp bình thường, biên và dữ liệu sai.
        Dự án được quản lý bằng GitHub Issues và commit gắn với từng issue cụ thể.
        Chạy kiểm thử bằng lệnh: mvn test.
        Mục tiêu là nâng cao kỹ năng kiểm thử tự động và làm việc với Git trong Java.
